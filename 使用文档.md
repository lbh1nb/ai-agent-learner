# AI Coding Learner 使用文档

> 版本 1.2.0 | 基于 Electron + Vue 3 的 AI 编程学习桌面应用

---

## 一、软件简介

AI Coding Learner 是一款帮助开发者学习如何使用 AI Agent（智能体）进行编程的桌面学习软件。它提供从零基础到进阶的完整课程体系，内置代码实操环境，并自动记录学习时长与进度。

### 核心功能

- **6 门系统课程**：涵盖 Agent 基础、Prompt Engineering、Tool Calling、工作流设计、多 Agent 协作、RAG 技术
- **15 个章节**：理论 + 实操结合，每章配有真实 Bilibili 教学视频 + 4 个配套学习资源链接
- **6 个实操任务**：5 个核心任务 + 1 个综合实战引导项目，Monaco 编辑器 + 分步引导（含预期效果 + 参考代码）+ 单元测试
- **Web Worker 代码沙箱**：安全执行 TypeScript 代码，支持类型剥离和测试用例验证
- **AI 助手对话**：支持接入真实 LLM（OpenAI 兼容 API），未配置时自动回退模拟模式
- **学习统计**：记录每日学习时长、连续学习天数、课程完成进度
- **竹绿色主题**：简洁美观的界面设计，护眼舒适

> **v1.2.0 更新**：①15 章节全部补充真实 B 站视频；②每章新增 4 个配套学习资源；③实操任务分步引导增强（预期效果 + 参考代码切换）；④修复右侧面板按钮被截断的布局问题
>
> **v1.1.0 更新**：实操实验室全面升级 — Monaco 编辑器、Web Worker 沙箱、5+1 分步引导任务、真实 LLM 接入

## 二、安装与启动

### 普通用户（推荐）

无需安装任何开发环境，三步即可使用：

1. **下载** — 前往 [GitHub Releases](https://github.com/lbh1nb/ai-agent-learner/releases) 下载最新版
   - `AI Coding Learner-1.2.0-win.zip`（解压版，推荐）
   - `AI Coding Learner 1.2.0.exe`（便携版，单文件）
2. **解压**（zip 版）— 将压缩包解压到任意目录，如桌面
3. **运行** — 双击 `AI Coding Learner.exe` 启动应用

> 便携版 `.exe` 双击即可运行，无需解压。

### 开发者（源码运行）

如需从源码运行或二次开发：

| 依赖 | 版本要求 |
|------|----------|
| Node.js | 18.x 或更高 |
| npm | 9.x 或更高 |
| 操作系统 | Windows / macOS / Linux |

```bash
git clone git@github.com:lbh1nb/ai-agent-learner.git
cd ai-agent-learner
npm install
npm run dev        # 开发模式
npm run pack       # 打包成可执行文件（输出到 release/）
```

启动后 Electron 窗口自动弹出，开发服务器运行在 `http://localhost:5173/`。

## 四、课程学习流程

### 4.1 浏览课程

进入 **课程学习** 页面，可看到 6 门课程卡片，每张卡片显示：

- 课程标题
- 难度等级（入门/进阶/高级）
- 课程分类
- 简短描述

点击卡片进入课程详情页。

### 4.2 课程详情

课程详情页展示：

- 课程介绍
- 章节列表（已完成章节标注对勾）
- 整体学习进度条

点击任意章节进入学习。

### 4.3 章节学习

章节学习页面分为三个区域：

```
┌──────────────┬─────────────────────────────────┐
│  左侧边栏    │         主内容区                  │
│              │                                 │
│  ← 返回课程  │  ┌─────────────────────────────┐ │
│              │  │  Bilibili 教学视频（内嵌）   │ │
│  章节列表    │  └─────────────────────────────┘ │
│  ├ 第1章 ✓  │                                 │
│  ├ 第2章 ✓  │  章节正文内容（Markdown 渲染）    │
│  ├ 第3章 ●  │  - 文字说明                      │
│  ├ 第4章    │  - 代码示例                      │
│  └ 第5章    │  - 架构图                        │
│              │                                 │
│              │  ┌─────────────────────────────┐ │
│              │  │ 📚 配套学习资源              │ │
│              │  │  • 官方文档链接              │ │
│              │  │  • 进阶教程                  │ │
│              │  │  • 开源项目                  │ │
│              │  │  • 博客文章                  │ │
│              │  └─────────────────────────────┘ │
│              │                                 │
│              │  ┌─────────────────────────────┐ │
│              │  │ [标记为已完成]  [上一章][下一章]│ │
│              │  └─────────────────────────────┘ │
└──────────────┴─────────────────────────────────┘
```

- **左侧章节列表**：点击任意章节可自由切换，已完成章节显示对勾
- **视频播放**：所有章节均内嵌 Bilibili 播放器，无需跳转浏览器
- **配套学习资源**：每章底部提供 4 个延伸学习链接（官方文档、进阶教程、开源项目、博客文章），点击在新窗口打开
- **进度追踪**：进入章节开始计时，每 30 秒自动保存学习时长
- **完成标记**：阅读完毕后点击"标记为已完成"，记录实际学习时长
- **章节导航**：使用"上一章"/"下一章"按钮顺序浏览

## 五、实操实验室

### 5.1 任务列表

进入 **实操实验室** 页面，查看 6 个编程任务（每个任务卡片显示难度、测试数量、引导步数）：

| 任务 | 难度 | 测试数 | 引导步数 | 说明 |
|------|------|--------|----------|------|
| Agent 指令构造器 | 简单 | 4 | 4 | 构造 Agent 系统指令 |
| 工具管理器 | 中等 | 4 | 4 | 注册和管理 Agent 工具 |
| 工作流执行器 | 困难 | 3 | 4 | 实现 Plan-Execute 工作流 |
| Agent 消息总线 | 困难 | 4 | 3 | 多 Agent 消息路由 |
| 文档检索器 | 困难 | 5 | 4 | RAG 文档检索与相似度排序 |
| 综合实战：Mini Agent | 困难 | 4 | 4 | 引导项目，组合检索 + 生成 |

> 带「引导项目」标签的任务是综合实战，会引导你从零构建一个完整的 Mini Agent。

### 5.2 任务界面

进入任务后，页面分为左右两栏：

```
┌────────────────────────┬──────────────────────────────┐
│  Monaco 代码编辑器      │  ┌────────────────────────┐  │
│                        │  │ 任务描述（Markdown）     │  │
│  // vs-dark 主题       │  │ 分步引导（进度条+步骤）  │  │
│  // TypeScript 高亮    │  │  步骤 1/4: 实现检索函数 │  │
│  // 自动补全           │  │  ✅ 预期效果: 返回数组   │  │
│                        │  │  [显示提示]             │  │
│  function create...    │  │  [View reference code]  │  │
│  // 格式化 | 重置      │  └────────────────────────┘  │
│                        │  ┌────────────────────────┐  │
│                        │  │ 测试结果面板（可滚动）  │  │
│                        │  │ ✓ 测试1: 有相关文档     │  │
│                        │  │ ✗ 测试2: 回答包含文档   │  │
│                        │  └────────────────────────┘  │
│                        │  ┌────────────────────────┐  │
│                        │  │ AI 助手 [真实LLM/模拟]  │  │
│                        │  │ [输入消息...] [发送]    │  │
│                        │  └────────────────────────┘  │
│                        │  ──────────────────────────  │
│                        │  [保存代码]  [运行测试]      │ ← 固定底部
└────────────────────────┴──────────────────────────────┘
```

> 右侧面板采用「上滚动 + 下固定」布局：任务面板、测试结果、AI 助手区域可纵向滚动，「保存代码」和「运行测试」按钮始终固定在底部可见，全屏/半屏模式下都不再被截断。

### 5.3 分步引导

每个任务包含 3-4 个步骤：
1. **步骤进度条**：顶部显示可点击的步骤点，蓝色=当前步，绿色=已完成
2. **步骤内容**：显示当前步骤的标题、详细描述
3. **预期效果**：每步底部绿色卡片明确告诉你完成后应得到什么结果
4. **提示**：点击"显示提示"查看代码思路（切换步骤时自动隐藏）
5. **参考代码**：点击"View reference code"展开/折叠该步骤的参考实现（暗色代码块，切换步骤时自动折叠）
6. **步骤导航**：上一步/下一步按钮

### 5.4 代码编辑器

- **Monaco Editor**：VS Code 同款编辑器，vs-dark 主题
- **TypeScript 语法高亮**：支持类型标注、泛型、接口
- **自动补全**：输入时弹出智能提示
- **格式化按钮**：一键格式化代码
- **重置按钮**：恢复为初始代码模板

### 5.5 运行测试

点击「运行测试」按钮，代码会在 **Web Worker 沙箱**中安全执行：
- 自动剥离 TypeScript 类型注解（interface、type、泛型等）
- 执行用户代码后，逐个运行测试用例
- 支持 4 种验证模式：精确匹配（equal）、包含匹配（contains）、数组长度（length）、异常抛出（throws）
- 测试结果面板显示每个用例的通过/失败状态、期望值和实际值
- 所有测试通过后才能提交任务

### 5.6 AI 助手

- **真实 LLM 模式**：在设置页配置 API Key 后，AI 助手调用真实大模型提供代码评审和答疑
- **模拟模式**：未配置 API Key 时自动使用本地模拟回复
- 界面顶部显示当前模式标签（「真实 LLM」或「模拟」）
- 可发送当前代码上下文给 AI，获得针对性建议

## 九、常见问题

**Q: 普通用户如何安装？**
A: 无需安装，从 [Releases](https://github.com/lbh1nb/ai-agent-learner/releases) 下载 zip 解压后双击 `AI Coding Learner.exe` 即可。

**Q: 学习时长不更新？**
A: 确保在章节页面停留超过 30 秒，系统每 30 秒自动记录一次。切换章节时也会保存。

**Q: 如何重置学习进度？**
A: 删除程序目录下的 `user-data/ai-coding-learner.db` 文件，重新启动即可。

**Q: 视频无法播放？**
A: 所有 15 章节均内嵌真实 Bilibili 视频播放器，正常情况下可在应用内直接观看。如遇黑屏，请检查网络连接，B 站视频需要联网播放。
