# AI Coding Learner 使用文档

> 版本 1.0.0 | 基于 Electron + Vue 3 的 AI 编程学习桌面应用

---

## 一、软件简介

AI Coding Learner 是一款帮助开发者学习如何使用 AI Agent（智能体）进行编程的桌面学习软件。它提供从零基础到进阶的完整课程体系，内置代码实操环境，并自动记录学习时长与进度。

### 核心功能

- **6 门系统课程**：涵盖 Agent 基础、Prompt Engineering、Tool Calling、工作流设计、多 Agent 协作、RAG 技术
- **15 个章节**：理论 + 实操结合，每章配有 Bilibili 教学视频
- **5 个实操任务**：内置代码编辑器，支持即时验证
- **AI 模拟对话**：模拟 Agent 交互，帮助理解 AI 编程思维
- **学习统计**：记录每日学习时长、连续学习天数、课程完成进度
- **竹绿色主题**：简洁美观的界面设计，护眼舒适

## 二、安装与启动

### 普通用户（推荐）

无需安装任何开发环境，三步即可使用：

1. **下载** — 前往 [GitHub Releases](https://github.com/lbh1nb/ai-agent-learner/releases) 下载最新版
   - `AI Coding Learner-1.0.0-win.zip`（解压版，推荐）
   - `AI Coding Learner 1.0.0.exe`（便携版，单文件）
2. **解压**（zip 版）— 将压缩包解压到任意目录，如桌面
3. **运行** — 双击 `AI Coding Learner.exe` 启动应用

> 便携版 `.exe` 双击即可运行，无需解压。

### 开发者（源码运行）

如需从源码运行或二次开发：

| 依赖 | 版本要求 |
|------|----------|
| Node.js | 18.x 或更高 |
| npm | 9.x 或更高 |
| 操作系统 | Windows / macOS / Linux |

```bash
git clone git@github.com:lbh1nb/ai-agent-learner.git
cd ai-agent-learner
npm install
npm run dev        # 开发模式
npm run pack       # 打包成可执行文件（输出到 release/）
```

启动后 Electron 窗口自动弹出，开发服务器运行在 `http://localhost:5173/`。

## 四、课程学习流程

### 4.1 浏览课程

进入 **课程学习** 页面，可看到 6 门课程卡片，每张卡片显示：

- 课程标题
- 难度等级（入门/进阶/高级）
- 课程分类
- 简短描述

点击卡片进入课程详情页。

### 4.2 课程详情

课程详情页展示：

- 课程介绍
- 章节列表（已完成章节标注对勾）
- 整体学习进度条

点击任意章节进入学习。

### 4.3 章节学习

章节学习页面分为三个区域：

```
┌──────────────┬─────────────────────────────────┐
│  左侧边栏    │         主内容区                  │
│              │                                 │
│  ← 返回课程  │  ┌─────────────────────────────┐ │
│              │  │  Bilibili 教学视频（内嵌）   │ │
│  章节列表    │  └─────────────────────────────┘ │
│  ├ 第1章 ✓  │                                 │
│  ├ 第2章 ✓  │  章节正文内容（Markdown 渲染）    │
│  ├ 第3章 ●  │  - 文字说明                      │
│  ├ 第4章    │  - 代码示例                      │
│  └ 第5章    │  - 架构图                        │
│              │                                 │
│              │  ┌─────────────────────────────┐ │
│              │  │ [标记为已完成]  [上一章][下一章]│ │
│              │  └─────────────────────────────┘ │
└──────────────┴─────────────────────────────────┘
```

- **左侧章节列表**：点击任意章节可自由切换，已完成章节显示对勾
- **视频播放**：部分章节内嵌 Bilibili 播放器；其他章节显示视频链接按钮
- **进度追踪**：进入章节开始计时，每 30 秒自动保存学习时长
- **完成标记**：阅读完毕后点击"标记为已完成"，记录实际学习时长
- **章节导航**：使用"上一章"/"下一章"按钮顺序浏览

## 五、实操练习

### 5.1 任务列表

进入 **实操练习** 页面，查看 5 个编程任务：

| 任务 | 难度 | 所属章节 |
|------|------|----------|
| 编写 Agent 指令 | 简单 | Prompt 实战练习 |
| 实现工具注册器 | 中等 | Tool Calling 实战 |
| 实现工作流引擎 | 困难 | 工作流实战 |
| 多 Agent 通信模拟 | 困难 | 多 Agent 实战 |
| RAG 检索器实现 | 困难 | RAG 实战 |

### 5.2 任务界面

进入任务后，页面分为三栏：

```
┌──────────────┬──────────────────┬──────────────────┐
│  任务面板     │    代码编辑器     │   AI 聊天助手    │
│              │                  │                  │
│  任务描述     │  // 在这里编写    │  Agent: 你好！   │
│  提示信息     │  // 你的代码      │  我来帮你完成    │
│  验证规则     │                  │  这个任务...     │
│              │  class Task {    │                  │
│  [提交验证]   │    ...          │  [输入消息...]    │
│  [保存代码]   │  }              │  [发送]          │
│              │                  │                  │
└──────────────┴──────────────────┴──────────────────┘
```

- **任务面板**：显示任务描述、验证规则
- **代码编辑器**：支持语法高亮，编写 TypeScript/JavaScript 代码
- **AI 聊天助手**：模拟 Agent 交互，可提问并获得指导
- **提交验证**：代码会自动检查是否满足要求（支持包含匹配、精确匹配、正则匹配）
- **保存代码**：随时保存当前代码，下次打开自动恢复

|------|------|------|

| 章节 | 类型 | 内容 |

## 九、常见问题

**Q: 普通用户如何安装？**
A: 无需安装，从 [Releases](https://github.com/lbh1nb/ai-agent-learner/releases) 下载 zip 解压后双击 `AI Coding Learner.exe` 即可。

**Q: 学习时长不更新？**
A: 确保在章节页面停留超过 30 秒，系统每 30 秒自动记录一次。切换章节时也会保存。

**Q: 如何重置学习进度？**
A: 删除程序目录下的 `user-data/ai-coding-learner.db` 文件，重新启动即可。

**Q: 视频无法播放？**
A: 部分章节是 Bilibili 搜索链接，点击后在浏览器中观看。直接视频支持内嵌播放。
